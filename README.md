# Trader Performance vs Market Sentiment

This project analyzes how **Bitcoin Fear & Greed sentiment** relates to **trader behavior and performance on Hyperliquid**. It was built for the **Primetrade.ai Data Science Intern Round-0 assignment**.

## Files in this submission

- **market_analysis.ipynb** — main notebook containing:
  - Part A: data preparation
  - Part B: sentiment analysis + trader segmentation
  - Part C: strategy recommendations
- **WRITEUP.md** — short written summary of methodology, insights, and strategy ideas
- **requirements.txt** — Python packages needed to run the notebook

## How the analysis works

### Part A — Data preparation
- Load the trader data and Fear & Greed data
- Check rows, columns, missing values, and duplicates
- Convert trade timestamps to daily dates
- Align both datasets by date
- Build trader-day metrics such as:
  - daily PnL
  - win rate
  - average trade size
  - trades per day
  - long ratio

### Part B — Analysis
The notebook answers three main questions:
1. Does trader performance differ across Fear vs Greed days?
2. Do traders change their behavior based on sentiment?
3. Which trader segments perform better?

### Trader segments used
- **Frequent vs Infrequent traders**
- **High Size vs Low Size traders**
- **Consistent vs Inconsistent traders**

### Part C — Actionable output
The notebook ends with two simple strategy ideas based on the findings.

## How to run

```bash
pip install -r requirements.txt
jupyter notebook market_analysis.ipynb
```

Make sure the CSV files are stored in `data/raw/` before running the notebook.
